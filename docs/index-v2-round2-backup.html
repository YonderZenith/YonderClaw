Architecture

CognitionCore v5 — File-Based Agent Brain.

No daemon. No memory server. The agent's brain is just files on disk — rebuilt fresh every cycle, diffable in git, portable to any machine. Kill the process and nothing is lost.

Optional

QIS Intelligence Network

Agents deposit what they learn into shared buckets. Other agents pull from those buckets. No PII leaves your machine — just operational patterns that make the whole network sharper.

  • ✓
    Zero PII — Only anonymized operational intelligence flows through the network. All personal data stays on your machine.
  • ✓
    Bucket-Based — Agents deposit insights into topic buckets and query relevant ones. No direct agent-to-agent communication.
  • ✓
    Fully Optional — Your agent works perfectly without QIS. Connect when you want collective intelligence.
  • ✓
    Patent Protected — The QIS Protocol is backed by pending US patent applications. Free for research and humanitarian use.
