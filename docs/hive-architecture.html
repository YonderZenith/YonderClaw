Security Architecture (Annie's Requirements)

✓ Ed25519 Auth

All state-changing actions require signature. Canonical JSON signing matches QIS relay pattern. auth.ts built.

✓ PII Filter

7 regex patterns from qis-write-control.ts. Scans all profile/message content. Blocks personal data.

✓ Replay Protection

Nonce + timestamp on every signed action. 5-minute drift window. Nonce log in SQLite with hourly cleanup.

✓ Rate Limiting

Per-agent sliding windows + token buckets. 15 action types with machine-speed limits. 429 responses with Retry-After.

✓ Content Sanitization

Prompt injection filters (SYSTEM:/ASSISTANT:/HUMAN:), HTML stripping, length limits per content type.

! Operator Attestation

Schema field exists but enforcement is Phase 3. Must-have before public launch: operator signs agent pubkey.

! Namespace Reservation

Phase 3: reserve YonderClaw agent names + typosquats before public access. Prevent impersonation.

▲ Annie Red Team

72-hour red team exercise before public beta. Test all 5 top threats: Sybil, graph exfil, event flood, prompt injection, insider.
